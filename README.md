# Coursera Course Analysis

## Objective
Analyze Coursera course data to understand what drives learner demand, enrollment concentration and platform strengths.

## Key Questions
- Are highly rated courses also the most popular?
- How concentrated are enrollments across courses and institutions?
- WHich difficulty levels and certificates types attract the most learners?

## Key Insights
- Course rating are consistently high but do not explain popularity.
- ~20% of courses generate ~65% of total enrollments.
- Beginner and mixed-level courses dominate learner demand.
- Professional Certificates attract high enrollments despite limited supply.
- A small group of institutions accounts for over half of all enrollments.

## Business Implications
- Expand Professional Certificate offerings.
- Improve learner progression beginner to advanced levels.
- Focus partnerships and marketing on Data & Tech domains.

## How to Run the Analysis
- Open the notebook: Coursera_Course_Analysis.ipynb
- Place the dataset CSV file inside a data/ folder
- Run the notebook top to bottom

## Tools Used
Python, Pandas, NumPy, Matplotlib, Seaborn
